response

绝大部分接口的响应类型为 application/json。此时，响应的JSON对象中都包含一个 code 字段，这用于标识请求成功与否：

  1. code 为 0，此时成功执行了请求。 对应的数据位于data字段。 它的具体类型跟接口有关。

    如：

    {"code": 0, "data": []}

  2. code 不为 0，此时请求未能正常完成。 对应的错误信息位于msg字段。 它的类型为object。

    如：

    {"code": 999, "msg": {"captcha": "缺少验证码"}}
